# Value Object

Value Object — это объект в домене, который:

- Не имеет идентичности (ID)
- Полностью определяется своими атрибутами
- Immutable (не изменяем после создания)
- Используется для представления значимых данных (например, `CategoryId`, `SessionId`, `DateTime`)

**Примеры:**

- `CategoryId`
- `SessionId`
- `DateTime`

Все Value Object должны быть валидабельны и сравниваемы по значению.
