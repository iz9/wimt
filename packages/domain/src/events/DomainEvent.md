# Domain Event

Domain Event — событие, которое произошло в домене и имеет значение для бизнес-логики.

- Отражает факт, который произошел
- Immutable
- Хранит только данные события
- Генерируется агрегатом
- Может использоваться для нотификаций, логирования, аналитики

**Примеры:**

- `CategoryCreated`
- `SessionStarted`
- `SessionPaused`
- `SessionStopped`
